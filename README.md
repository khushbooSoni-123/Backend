# 🚀 Node.js CRUD API (MongoDB)

This project is a complete backend CRUD API built using:
- Node.js
- Express.js
- MongoDB
- Mongoose

---

## 📁 Project Structure
mongodb/
│
├── config/
│ └── db.js # MongoDB connection
│
├── models/
│ └── user.model.js # User schema
│
├── server.js # Main server file
├── package.json
└── .gitignore


---

## ⚙️ Features

✅ Create User (POST)  
✅ Get All Users (GET)  
✅ Get Single User (GET by ID)  
✅ Update User (PUT)  
✅ Delete User (DELETE)

---

## 🔗 API Endpoints

### CREATE USER
POST `/users`

### GET ALL USERS
GET `/users`

### GET SINGLE USER
GET `/users/:id`

### UPDATE USER
PUT `/users/:id`

### DELETE USER
DELETE `/users/:id`
